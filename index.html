Introduction

       Welcome to visit Bioinformatic Tools and Pipelines Laboratory of Yersinia pestis branch of ICDC !

       Bioinformatic Tools and Pipelines Laboratory is a repository for lab members and collaborators to share useful scripts and applications. We aims to help micriobiologists and molecular epidemilogists with or without programming skills to analysis genes or genomes sequences,especailly in the field of pathogen source tracing and typing.

  • Basic Sequence Editing scripts;
  • Common Sequence Alignment scripts;
  • Graphic display pipelines;
  • Bacterial molecular typing pipelines;
    • Mutilocus sequence typing(MLST) pipelines
    • Whole genome Mutilocus sequence typing(wgMLST) pipelines
    • Mutilocus Variable-number tandem repeat Analysis pipelines
    • Clustered regularly interspaced short palindromic repeats analysis(CRISPR) pipelines
    • Single nucleotide polymorphism (SNP) analysis pipelines
  • Self-defined applications.
